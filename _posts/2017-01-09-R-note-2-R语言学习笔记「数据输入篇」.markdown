#### 手工录入

```R
mydata <- data.frame(
  age=numeric(0),
  gender=character(0),
  weight=numeric(0)
)

mydata <- edit(mydata)
```

#### 导入CSV格式

```R
csvdata <- read.table(
  'data.csv', 
  header = TRUE,
  sep = ',',
  row.names = '病人编号'
)
```

#### 导入Excel格式

```R
library(RODBC)
channel <- odbcConnectExcel('data.xls')
mydata2 <- sqlFetch(channel,'Sheet1')
odbcClose(channel)
```
